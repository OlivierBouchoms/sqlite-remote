# SQLite Remote - Backend

## Prerequisites

- .NET 8

## Getting started

```shell
dotnet restore
dotnet build
dotnet run --environment Development
```

The API will be available at [http://localhost:7000](http://localhost:7000). 
API docs are available at [http://localhost:7000/swagger/index.html](http://localhost:7000/swagger/index.html).

## Used tools

* SSH.NET (SSH client)
* Swashbuckle (API documentation)
